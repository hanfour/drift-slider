Fade Effect

淡入淡出效果

Smooth cross-fade transition using effect: 'fade' with EffectFade module.

使用 effect: 'fade' 搭配 EffectFade 模組的平滑交叉淡入轉場。

  • Night Sky
  • Violet
  • Teal
  • Deep Blue

Source Code

js 
var DS = DriftSlider;
var slider = new DS.DriftSlider('.drift-slider', {
  modules: [DS.Navigation, DS.Pagination, DS.EffectFade, DS.A11y],
  effect: 'fade',
  speed: 600,
});
