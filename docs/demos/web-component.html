Web Component

Web Component 元件

Use DriftSlider as a native HTML custom element — drop in <drift-slider> with attributes, no JavaScript boilerplate required.

以原生 HTML 自訂元素使用 DriftSlider — 只需加入帶有屬性的 <drift-slider>，無需撰寫任何 JavaScript 樣板程式碼。

Current Slide: 1 / 5 目前投影片：1 / 5 Progress: 0% 進度：0%

Source Code

html 
<!-- CDN: auto-registers <drift-slider> and injects CSS -->
<script src="https://unpkg.com/drift-slider-element"></script>

<drift-slider loop navigation pagination slides-per-view="1" speed="500">
  <img src="https://picsum.photos/seed/wc1/900/340" alt="Slide 1">
  <img src="https://picsum.photos/seed/wc2/900/340" alt="Slide 2">
  <img src="https://picsum.photos/seed/wc3/900/340" alt="Slide 3">
  <img src="https://picsum.photos/seed/wc4/900/340" alt="Slide 4">
  <img src="https://picsum.photos/seed/wc5/900/340" alt="Slide 5">
</drift-slider>

<script>
  var el = document.querySelector('drift-slider');
  el.addEventListener('drift:slidechange', function(e) {
    document.getElementById('current-index').textContent = e.detail.activeIndex + 1;
    var pct = Math.round(e.detail.progress * 100);
    document.getElementById('current-progress').textContent = pct + '%';
  });
</script>

Bundler Usage

打包工具用法

js 
import 'drift-slider-element/define';
import { registerModules } from 'drift-slider-element';
import { Navigation, Pagination } from 'drift-slider/modules';
import 'drift-slider/css/bundle';

registerModules({ Navigation, Pagination });
